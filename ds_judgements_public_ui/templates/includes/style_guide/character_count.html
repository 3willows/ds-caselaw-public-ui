Character count

Help users know how much text they can enter when there is a limit on the number of characters.

You can enter up to 100 characters

Usage:

{# fmt:off #} <div class="govuk-character-count" data-module="govuk-character-count" data-maxlength="100"> <div class="govuk-form-group"> <textarea class="govuk-textarea govuk-js-character-count" id="with-hint" name="withHint" rows="5" aria-describedby="with-hint-info with-hint-hint"> </textarea> </div> <div id="with-hint-info" class="govuk-hint govuk-character-count__message"> You can enter up to 100 characters </div> </div> {# fmt:on #}
